 	AWS Certified Cloud Practitioner – 10-Day Roadmap (CLF-C02)



 	Day 1: 	Introduction to Cloud \& AWS Basics,What is Cloud Computing? Cloud models: IaaS, PaaS, SaaS, Deployment models: Public, Private, Hybrid, Introduction to AWS, Overview of AWS 		global infrastructure: Regions, Availability Zones, Edge Locations, AWS core services categories: Compute, Storage, Networking, Database

 	Time: ~2 hrs

 	Practice: Create a free-tier AWS account. Explore AWS Management Console, and list out 10 services you see on the home page.



 	Day 2: 	AWS Core Services – Compute, Storage, Database, Compute, Amazon EC2 (basic concept only), AWS Lambda (serverless concept), Storage, Amazon S3, Storage classes, durability, 		EBS, EFS, Glacier (brief overview), Databases, Amazon RDS and DynamoDB overview

 	Time: ~2.5 hrs

 	Practice: Use the Console to create an S3 bucket and upload a file. Explore EC2 launch wizard (no need to launch).



 	Day 3: 	AWS Networking and Content Delivery, Networking Basics, What is a VPC (high-level only), Internet Gateway, NAT Gateway, Subnets (simplified view), Content Delivery, 			CloudFront CDN concept, Route 53, DNS in AWS,

 	Time: ~2 hrs

 	Practice: Use AWS Global Infrastructure page to explore regions and availability zones. Try simulating CloudFront setup in Console.



 	Day 4: 	Security and Compliance, Shared Responsibility Model, AWS Identity and Access Management (IAM), Users, Groups, Roles, Policies, MFA, IAM Best Practices, Compliance 			Programs, HIPAA, GDPR, ISO, PCI DSS, AWS Organizations (basic overview)

 	Time: ~2 hrs

 	Practice: Create an IAM user, apply a policy, and enable MFA. Explore IAM roles and policies.



 	Day 5: 	Billing, Pricing, and Support Plans, AWS Pricing Models, Pay-as-you-go, Free tier, On-demand, Reserved, Spot, Cost Estimation Tools, AWS Pricing Calculator, AWS Cost , 		Explorer, Support Plans, Basic, Developer, Business, Enterprise

 	Time: ~2 hrs

 	Practice: Use the AWS Pricing Calculator to estimate EC2 and S3 costs. Compare AWS Support Plans in the documentation.



 	Day 6: 	Cloud Concepts and AWS Value Proposition, Benefits of Cloud Computing, Agility, Scalability, Elasticity, Global Reach, Cloud Adoption Framework, Perspectives: Business, 		People, Governance, Platform, Security, Operations, Well-Architected Framework (Brief)

 	Time: ~2 hrs

 	Practice: List 5 reasons why a company should migrate to AWS. Skim through the Well-Architected Framework whitepaper (just the summary).



 	Day 7: 	Monitoring and Governance, AWS CloudWatch, Monitoring and Alarms, AWS CloudTrail, Logging AWS account activity, AWS Trusted Advisor, Recommendations for cost optimization, 		security, performance

 	Time: ~2 hrs

 	Practice: Enable CloudTrail and view recent events. Set a simple CloudWatch alarm on CPU usage (you can simulate without running a workload).



 	Day 8: 	Application Integration \& Deployment, Messaging Services, Amazon SQS, SNS (basic understanding), Deployment Tools, AWS Elastic Beanstalk (PaaS concept), AWS CodePipeline 		(brief mention)

 	Time: ~2 hrs

 	Practice: Simulate SNS topic creation and subscription in Console.



 	Day 9: 	Review Day + Whitepapers, Review of All Key Services, Compute, Storage, Database, IAM, Billing, CloudWatch, CloudTrail, Whitepapers (Skim These), AWS Well-Architected 			Framework, AWS Overview Whitepaper, AWS Security Best Practices

 	Time: ~3 hrs

 	Practice: Flashcards, summarize in your own words what each core service does.



 	Day 10: Practice Exam \& Exam Strategy, Take Official Practice Test, Use AWS-provided questions or external platforms like Tutorials Dojo, Whizlabs, or SkillCertPro, Review Missed 		Questions, Understand why an answer is correct or wrong, Exam Strategy, Eliminate wrong answers, Flag tricky questions, don’t dwell too long, Read all question keywords: 		“MOST cost-effective”, “BEST solution”, “MINIMIZE effort”, etc.



 	Time: ~3-4 hrs

 	Practice: Full practice exam under timed conditions. Review every question afterward.
